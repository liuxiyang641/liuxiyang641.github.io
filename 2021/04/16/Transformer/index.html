Attention Is All You Need

NIPS 2017

жң¬ж–ҮжҸҗеҮәдәҶдёҖдёӘж–°зҡ„з®ҖеҚ•зҡ„зҪ‘з»ңз»“жһ„пјҢTransformerпјҢеҸӘдҫқиө–дәҺжіЁж„ҸеҠӣжңәеҲ¶гҖӮ

еҸҜеҸӮиҖғзҡ„еҚҡе®ў

1 Introduction

RNNзӯүжЁЎеһӢе·Із»ҸеҸ–еҫ—дәҶеҫҲеӨ§зҡ„жҲҗеҠҹпјҢдҪҶжҳҜи®Ўз®—еҫӘзҺҜзҪ‘з»ңзҡ„д»Јд»·йҖҡеёёдјҡдёҺиҫ“е…Ҙиҫ“еҮәеәҸеҲ—зҡ„з¬ҰеҸ·дҪҚзҪ®зҙ§еҜҶзӣёе…ігҖӮеәҸеҲ—зҡ„з¬ҰеҸ·дҪҚзҪ®еҶіе®ҡдәҶеҫӘзҺҜзҪ‘з»ңдёӯиҫ“е…Ҙзҡ„жӯҘж•°пјҢеҜјиҮҙеҫҲйҡҫе№¶иЎҢеҢ–пјҢе°ұеңЁж—¶й—ҙе’Ңи®Ўз®—иө„жәҗпјҲеҶ…еӯҳпјүдёҠйҷҗеҲ¶дәҶжЁЎеһӢзҡ„и®ӯз»ғгҖӮ

жіЁж„ҸеҠӣжңәеҲ¶еңЁsequence modeling е’Ң transduction modelsд»»еҠЎдёҠе·Із»ҸеҸ–еҫ—дәҶеҫҲеӨ§зҡ„жҲҗе°ұпјҢдҪҶжҳҜеҫҲе°‘жңүжЁЎеһӢдјҡе°ҶжіЁж„ҸеҠӣжңәеҲ¶дёҺRNNиҒ”зі»еңЁдёҖиө·гҖӮ

жң¬ж–Үе°ұжҸҗеҮәдәҶдёҖдёӘж–°зҡ„ж–№жі•TransformerпјҢйҒҝе…ҚдәҶеҫӘзҺҜпјҢзӣёеҸҚзҡ„жҳҜеҹәдәҺжіЁж„ҸеҠӣжңәеҲ¶е®Ңе…Ёдҫқиө–дәҺиҫ“е…Ҙе’Ңиҫ“еҮәзҡ„ж•ҙдҪ“гҖӮ

Transformer is the п¬Ғrst transduction model relying entirely on self-attention to compute representations of its input and output without using sequencealigned RNNs or convolution.

2 Model Architecture

ж•ҙдёӘTransformerзҡ„з»“жһ„жҳҜencoderе’Ңdecoderзҡ„з»“жһ„пјҢеҰӮдёӢеӣҫжүҖзӨәгҖӮ

ж•ҙдҪ“е ҶеҸ 6еұӮencoderпјҢз„¶еҗҺз»ҸиҝҮ6еұӮзҡ„decoderгҖӮ6еұӮencoderзҡ„жңҖз»Ҳиҫ“е…Ҙдјҡиҫ“е…ҘеҲ°жҜҸдёҖеұӮзҡ„decoderдёӯгҖӮ

жҜҸдёҖеұӮencoderеҢ…жӢ¬дёӨдёӘsublayerпјҢдёҖдёӘжҺҘеҸ—inputзҡ„иҫ“е…ҘпјҢз„¶еҗҺз»ҸиҝҮ\(h\)дёӘеӨҡеӨҙжіЁж„ҸеҠӣпјҢеҠ дёҠеҺҹжқҘзҡ„иҫ“е…ҘпјҢд№ӢеҗҺnormпјҢз¬¬дәҢеұӮз»ҸиҝҮдёҖдёӘеүҚйҰҲзҪ‘з»ңпјҢиҝҳжҳҜеҠ дёҠеҺҹжқҘзҡ„иҫ“е…ҘпјҢз»ҸиҝҮnormгҖӮencoderзҡ„еҲқе§Ӣиҫ“е…ҘжҳҜе…ЁйғЁзҡ„еҺҹе§Ӣиҫ“е…ҘгҖӮ

жҜҸдёҖеұӮзҡ„decoderеҢ…жӢ¬дёүдёӘsublayerпјҢжңүдёӨдёӘдёҺencoderдёҖж ·пјҢдҪҶжҳҜеӨҡдәҶдёҖеұӮдјҡжҺҘеҸ—encoderзҡ„иҫ“еҮәдҪңдёәkeysе’ҢvaluesгҖӮdecoderзҡ„еҲқе§Ӣиҫ“е…ҘжҳҜ\(t-1\)ж—¶еҲ»зҡ„йў„жөӢз»“жһңд»ҘеҸҠencoderзҡ„иҫ“еҮәгҖӮ

дёӢйқўиҜҰз»Ҷи§Јжһҗпјҡ

2.1 Multi-head attention

йҰ–е…Ҳи®Ўз®—еҚ•дёӘattentionпјҢдҪҝз”ЁqueryпјҢkeysе’ҢvaluesжқҘи®Ўз®—гҖӮдҪҝз”Ёqueryе’Ңе…¶е®ғжүҖжңүembeddingзҡ„keysи®Ўз®—еҮәжқғеҖјпјҢз„¶еҗҺдёҚеҗҢзҡ„жқғеҖјдёҺvaluesзӣёд№ҳжұӮе’ҢгҖӮquerysе’Ңkeysзҡ„з»ҙеәҰжҳҜ\(d_k\)пјҢvaluesзҡ„з»ҙеәҰжҳҜ\(d_v\)гҖӮ \[ Attention(Q,K,V)=softmax(\frac{QK^T}{\sqrt{d_k}})V \] йңҖиҰҒжіЁж„Ҹзҡ„жҳҜе…¶дёӯеӨҡдәҶдёҖдёӘеёёйҮҸ\(\sqrt{d_k}\)пјҢиҝҷжҳҜеӣ дёәдҪңиҖ…еңЁе®һйӘҢдёӯеҸ‘зҺ°еҠ е…ҘйҷӨж•°еёёйҮҸзҡ„ж•ҲжһңеҫҲеҘҪпјҢеҺҹеӣ еҸҜиғҪжҳҜеӣ дёәsoftmaxзҡ„иҫ“е…ҘеңЁеҖјиҫғеӨ§зҡ„ж—¶еҖҷжўҜеәҰдјҡеҸҳе°ҸпјҢеӣ жӯӨеҠ е…ҘдәҶдёҖдёӘйҷӨж•°еёёйҮҸеҮҸе°ҸеҖјгҖӮ

и®Ўз®—е®ҢжҲҗеҚ•дёӘattentionд№ӢеҗҺпјҢеҶҚи®Ўз®—еӨҡеӨҙжіЁж„ҸеҠӣпјҢжӢјжҺҘиө·жқҘд№ӢеҗҺеҶҚд№ҳд»ҘдёҖдёӘжқғеҖјзҹ©йҳөгҖӮ \[ MultiHead(Q,K,V)=Concat(head_1,\dots,head_h)W^O \] еңЁе®һи·өдёӯпјҢдҪҝз”ЁдәҶ8дёӘеӨҙпјҢжҜҸдёӘз»ҙеәҰ64пјҢдёҖе…ұз»ҙеәҰ512гҖӮ

decoderзҡ„з»“жһ„дёҺencoderзұ»дјјпјҢдҪҶжҳҜе®ғеӨҡдәҶдёҖеұӮencoderе’ҢdecoderгҖӮ

2.2 self-attention

з¬¬дёҖжӯҘпјҡеҜ№дәҺиҫ“е…Ҙзҡ„жҜҸдёҖдёӘvectorеҲӣе»ә3дёӘж–°зҡ„vectorпјҢ a Query vector, a Key vector, and a Value vectorгҖӮ

з¬¬дәҢжӯҘпјҡи®Ўз®—еҚ•дёӘзҡ„scoreпјҢжҜ”еҰӮиҜҙи®Ўз®—з¬¬дёҖдёӘиҜҚThinkingпјҢйңҖиҰҒи®Ўз®—ж•ҙдёӘеәҸеҲ—еҪ“дёӯжүҖжңүзҡ„vectorеҜ№дәҺThinkingзҡ„vectorзҡ„йҮҚиҰҒзЁӢеәҰпјҢдҪҝз”ЁThinkingзҡ„query vectorе’Ңе…¶е®ғжүҖжңүзҡ„key vectorеҒҡdot productгҖӮ

з¬¬дёүжӯҘдёҺз¬¬еӣӣжӯҘпјҡе®һйҷ…жҳҜеҪ’дёҖеҢ–socreпјҢзӣёеҪ“дәҺдә§з”ҹrelative scoreгҖӮ

з¬¬дә”жӯҘпјҡеҗ„дёӘword vectorдёҺrelative scoreзӣёд№ҳпјҢжұӮе’ҢгҖӮиҝҷж ·зј–з ҒеҗҺзҡ„жҹҗдёӘдҪҚзҪ®зҡ„ж–°зҡ„embeddingжҳҜз”ұеүҚдёҖжӯҘжүҖжңүиҫ“е…Ҙзҡ„embeddingе…ұеҗҢеҶіе®ҡзҡ„гҖӮ

з¬¬е…ӯжӯҘпјҡзҹ©йҳөеҪўејҸзҡ„е®һйҷ…и®Ўз®—жғ…еҶө
